Download GoPHPServer

GoPHPServer is available for Windows, macOS, and Linux. Choose the appropriate download for your operating system below.

Windows

Download the Windows binary and extract it to use GoPHPServer on Windows.

Download for Windows
# Extract the zip file
# Run the executable
gophpserver.exe

macOS

Download the macOS binary and extract it to use GoPHPServer on macOS.

Download for macOS
# Extract the archive
tar -xzf gophpserver-darwin-amd64.tar.gz

# Make it executable
chmod +x gophpserver

# Run the server
./gophpserver

Linux

Download the Linux binary and extract it to use GoPHPServer on Linux.

Download for Linux
# Extract the archive
tar -xzf gophpserver-linux-amd64.tar.gz

# Make it executable
chmod +x gophpserver

# Run the server
./gophpserver

Build from Source

If you prefer to build GoPHPServer from source, you can clone the repository and build it yourself:

# Clone the repository
git clone https://github.com/chandan1106/gophpserver.git
cd gophpserver

# Build the server
make build

# Run the server
./build/gophpserver

Docker Image

You can also use the Docker image to run GoPHPServer:

# Pull the image
docker pull chandan1106/gophpserver:latest

# Run the container
docker run -d -p 8080:8080 -v $(pwd)/public:/app/public chandan1106/gophpserver
