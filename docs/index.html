Features

Lightweight & Fast

Built with Go's efficient HTTP server implementation, GoPHPServer is lightweight and performs exceptionally well for development and small to medium production sites.

PHP Support

Execute PHP scripts directly using the PHP interpreter installed on your system. No need for PHP-FPM or other complex setups.

Simple Configuration

Configure the server with simple command-line flags. No complex configuration files or settings to manage.

Security Features

Includes security headers by default to help protect your applications from common web vulnerabilities.

Docker Support

Run GoPHPServer in Docker with the included Dockerfile and docker-compose configuration for easy containerization.

Cross-Platform

Works on Windows, macOS, and Linux. Build once, run anywhere with Go's cross-compilation capabilities.

Installation

Option 1: Download Pre-built Binary

Download the binary for your platform from our GitHub releases page:

# Download the binary for your platform
# Linux
wget https://github.com/chandan1106/gophpserver/releases/download/v1.0.0/gophpserver-linux-amd64.tar.gz

# Extract the archive
tar -xzf gophpserver-linux-amd64.tar.gz

# Make it executable
chmod +x gophpserver

Option 2: Build from Source

# Clone the repository
git clone https://github.com/chandan1106/gophpserver.git
cd gophpserver

# Build the server
make build

Running the Server

# Start the server
./gophpserver

# Or with custom options
./gophpserver -port 8000 -docroot /path/to/your/website
